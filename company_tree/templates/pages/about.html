Тестовое приложение для вывода информации о подразделениях с большим количеством уровней вложенности и более 50 тысяч сотрудников. Реализован на базе Django 4.0 с использованием библиотеки django-mptt. Дизайн оформления — Twitter Bootstrap 5.


Для заполнения базы данных написан специальный Python-скрипт. Относительно структуры компании предполагается общий случай: один и тот же сотрудник может работать в нескольких подразделениях, и на каждом уровне может быть несколько сотрудников.
